Welcome to the Home page for LEM Security’s “Project Guardian”! From this web GUI you will be able to change the rules for the firewall by clicking the “Administration” tab as well as check the IDS logs by clicking the “View Logs” tab. There is also a “Help Page” where you can go if you need assistance with adjusting rules or other kinds of technical help.
